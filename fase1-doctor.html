Verificando…

Rodando o diagnóstico.

…SDK do Firebase carregado
…Authentication inicializado no Console
Falta ligar o serviço. Console → Authentication → Começar. Erro recebido: auth/configuration-not-found.
…Provedor Email/Senha habilitado
O provedor está desligado. Console → Authentication → Sign-in method → habilitar Email/Senha. Erro: auth/operation-not-allowed.
…Domínio autorizado
Este domínio não está autorizado. Console → Authentication → Settings → Authorized domains → adicionar este domínio (e localhost). Erro: auth/unauthorized-domain.
